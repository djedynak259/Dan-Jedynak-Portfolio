Dan Jedynak

Front End Developer - UI/UX

Here's a story about what I do:

My software career began with a position in customer experience. As I interacted directly with thousands of customers, I studied how they used software. I learned the difference between a good user experience and a great user experience. Because I’m passionate about whatever I’m involved in, I felt my customers’ pain points. I decided to learn to code to fix them more directly. Within 3 months I was given access to the code base and was improving the UI on my own time. Each time I made an improvement and presented it to the product manager, the design change was approved and my code was pushed to production. I took on these extra challenges because I'm deeply committed to continual learning and driven to make a difference.
